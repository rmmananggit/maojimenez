##

QR CODE LOGIN Using PHP & MySQL | SCAN QR CODE LOGIN 2021
------------

0. Open your localhost provider. - XAMPP/WAMP server.

1. First, create a QRCode Text using QRCode Generator in any free website generator.

2. then save the QRCode pic/png format.

3. Save the QRCode Text in the database, table of users.

4. Then try to login and scan the QRCode pic/png through login form of your project.
	*If QRCode doesn't exist or empty, the error will show up and will not be able to enter the homepage.

5. That's all. :)
